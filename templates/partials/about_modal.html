About nl2sql-366

nl2sql-366 is a hybrid natural-language-to-SQL tool. It combines a machine-learning intent classifier with rule-based attribute matching to turn plain-English questions into real SQL queries — no coding required.

  • Upload any CSV or Excel file — columns and sample values are read automatically.
  • Type a question like "average salary by district".
  • The AI detects your intent, matches the right columns, and builds a safe SQL query.
  • Results appear instantly. No data ever leaves this app — everything runs locally.
